LED Control System Model

Author: Anas Alhashimi

Description: This project demonstrates an intelligent LED control system for efficient power management and dynamic lighting based on sensor inputs.

Part 1: Learning the Model

In this phase, we first implement a generic Arduino code for data acquisition. Then, we use a MATLAB script to record input/output data, analyze the system response, and estimate the model order and parameters.

Part 2: Building the Controller

Using the best-estimated model, we design and implement a PID controller to maintain the light intensity at a fixed level.

Single LED / Single LDR System

  • Microcontroller: Arduino / ESP32
  • Sensor: LDR (Light Dependent Resistor)
  • Control Algorithm: Model-based PID Controller
  • Programming Language: C++ (Arduino IDE) & MATLAB
  • Data Processing: MATLAB for system identification

Extending to 2 LEDs / 2 LDRs System

After verifying the control performance for a single LED-LDR system, the same methodology is applied to a system with two LEDs and two LDRs, allowing for more complex control and lighting optimization.

Challenges & Solutions

One major challenge was ensuring smooth dimming transitions. This was resolved using PWM control with adaptive filtering to reduce flickering.
